## Setup
Renomeie **.env.example** para **.env** e adicione seu [github TOKEN](https://github.com/settings/tokens) e github EMAIL</p>  

Agora inicie o app com o gerenciador de pacotes [yarn](https://yarnpkg.com/)
```
yarn
yarn android
yarn start
```
